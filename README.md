### **create a repo from cli**
```
git init  
git commit -m "first commit"  
git branch -M main  
git remote add origin https://github.com/elevavmars/[repo].git  
git push -u origin main  
```

### push an existing repo from the cli
```
git remote add origin https://github.com/elevavmars/[repo].git  
git branch -M main  
git push -u origin main  
```
